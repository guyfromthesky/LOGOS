# LOGOS Documentation

This directory contains the authoritative documentation for the LOGOS protocol.

## Structure

*   `WHITEPAPER.md`: The original whitepaper defining the core concepts.
*   `rfc/`: Request for Comments (RFCs) decomposing the whitepaper into specific, immutable specifications.
*   `diagrams/`: Mermaid diagrams used in the documentation.
*   `meta/`: Meta-documentation including terminology and invariants.

## RFC Series

*   **RFC-0001**: Core Protocol
*   **RFC-0002**: Graph Substrate
*   **RFC-0003**: Threat Model
*   **RFC-0004**: Ledger & Governance
*   **RFC-0005**: Sediment vs Bedrock
*   **RFC-0006**: Comparative Analysis
