# RFC-0005: Sediment and Bedrock Model

*   **Status**: Draft
*   **Category**: Lifecycle

## 1. Layer Definitions

*   **Sediment Layer**: Mutable sandbox for autonomous AI experimentation.
*   **Bedrock Layer**: Locked execution layer for human-verified logic.

## 2. Node Lifecycle

1.  AI proposes Node (Sediment)
2.  Node executes under constraints
3.  Human audits behavior
4.  Node promoted to Bedrock or deprecated

## 3. Promotion Rules

*   Requires successful execution history
*   Requires human approval
*   Recorded in Ledger
